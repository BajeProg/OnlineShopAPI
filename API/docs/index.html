Основная информация

Автор Владимир Баженов
Задача Разработать API, которая дает возможность сторонним приложениям, сайтам и т.д. добавлять, удалять, изменять и просматривать данные из базы данных интернет-магазина
Бизнес постановка API нужно для обращения приложения к базе данных интернет-магазина. Это обеспечит пользователям удобство в пользовании услуг магазина и увеличит продажи

Входные параметры

Параметр Тип данных Обязательность Описание
token string да Токены выдаются разработчикам для идентификации их приложения
type string да Этот параметр описывает конкретные функции API
login string нет Этот параметр содержит логин пользователя
pass string нет Этот параметр содержит пароль пользователя
userid int нет Этот параметр содержит ID пользователя
url string нет Этот параметр содержит адрес на изображение
name string нет Этот параметр содержит название товара
desc string нет Этот параметр содержит описание товара
price int нет Этот параметр содержит цену товара
productid int нет Этот параметр содержит ID товара
address string нет Этот параметр содержит адрес клиента для доставки
orderid int нет Этот параметр содержит ID заказа
